# Plan 27.0: Entrenamiento YOLO Segmentation con Datos Manuales (V1 Optimized)

## 1. Objetivo
Entrenar un modelo de segmentación (YOLO 26) utilizando exclusivamente el dataset curado y etiquetado manualmente que se encuentra en `segmented/`. El fin es lograr una precisión del 100% en la detección de moscas sobre ROIs con fondo negro.

## 2. Origen de Datos (Dataset Preparado)
*   **Ubicación:** `dataset_moscas_v2/final_dataset_moscas/train/segmented/`
*   **Contenido:** Imágenes y etiquetas (polígonos) extraídas del archivo `moscasV1YOLOSegmentation1.0.zip`.
*   **Estado:** El dataset ya está organizado con la estructura `images/train` y `labels/train`.
*   **Clase:** `0: mosca`.

## 3. Configuración de Ultralytics (`data.yaml`)
El archivo ya ha sido configurado en la carpeta del dataset con los siguientes parámetros:
```yaml
path: /data/estudiantes/vacas/dataset_moscas_v2/final_dataset_moscas/train/segmented
train: images/train
val: images/train  # Usamos el mismo set para validación rápida o split interno
names:
  0: mosca
```

## 4. Pasos de Ejecución

### Paso 1: Lanzamiento del Entrenamiento (`27.1.entrenar_moscas_v1.py`)
Crearemos un script de entrenamiento que utilice el modelo YOLO 26.
*   **Modelo Base:** `yolo26n-seg.pt` (Nano para velocidad) o `yolo26m-seg.pt` (Medium para precisión).
*   **Configuración:**
    *   `imgsz=640`
    *   `epochs=100`
    *   `batch=16`
    *   `device=0` (O las GPUs disponibles).
    *   `retina_masks=True` (Para mayor definición en los segmentos pequeños de las moscas).

## 5. Evaluación de Resultados
1.  **Métricas de Máscara:** Monitorear Precision y Recall.
2.  **Validación Visual:** Comprobar que el modelo ignore manchas de piel y detecte moscas incluso en condiciones de baja luz/fondo negro.
